# Arquivos JSON

- Todos os arquivos devem utilizar UTF-8 e JSON válido, sem comentários.
- IDs existentes não devem ser alterados.
- Relações entre coleções devem utilizar IDs.
- Cada nova coleção deve ser incluída em `manifest.json`.
- As coleções desta etapa estão vazias e não representam um banco implementado.
